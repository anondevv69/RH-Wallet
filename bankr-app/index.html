How to connect Robinhood Agentic
  1. 1 On your computer, run (copied when you click Connect):
    curl -fsSL https://raw.githubusercontent.com/anondevv69/RH-Wallet/main/scripts/rh-connect.sh | bash
  2. 2 Sign in with Robinhood and tap Allow — this works with localhost only.
  3. 3 After login you'll see your access token. Copy it.
  4. 4 In Bankr: Settings → Env Vars → + Add
    Key: AGENTIC_TOKEN   Value: paste token
  5. 5 Add MCP server in Bankr settings:
    Name: robinhood-agentic
    URL: …
    Transport: Streamable HTTP
    Header: Authorization → Bearer {{AGENTIC_TOKEN}}
  6. 6 Come back and click Refresh to verify. Then ask Bankr: "What is my Robinhood Agentic buying power?"
