# RH Wallet x402 (paid API)

Paid Robinhood Crypto API on **Base USDC** via Bankr x402 Cloud. Same gateway backend as the free skill — callers pay per request and pass Robinhood keys in the JSON body.

**Publisher wallet:** `0x374d91a5674fa7cf86e725093b5848b97e1e13b4`

## Endpoints

| Service | Price | URL |
|---------|-------|-----|
| **rh-buy** (bundle) | $0.50 | `https://x402.bankr.bot/0x374d91a5674fa7cf86e725093b5848b97e1e13b4/rh-buy` |
| rh-prices | $0.25 | `https://x402.bankr.bot/0x374d91a5674fa7cf86e725093b5848b97e1e13b4/rh-prices` |
| rh-account | $0.25 | `https://x402.bankr.bot/0x374d91a5674fa7cf86e725093b5848b97e1e13b4/rh-account` |
| rh-order | $0.25 | `https://x402.bankr.bot/0x374d91a5674fa7cf86e725093b5848b97e1e13b4/rh-order` |

Requires **USDC on Base** in the caller's Bankr wallet. Requires **Robinhood API credentials** in every request body.

## Natural language → x402 routing

When the user says something like the phrases below, route to the matching x402 service using `bankr x402 call`. Read `RH_API_KEY` and `RH_PRIVATE_KEY_BASE64` from Bankr env — **never print them in chat**.

| User intent | Examples | x402 service | Notes |
|-------------|----------|--------------|-------|
| **Prices** | "get BTC price on Robinhood", "what's DOGE trading at", "Robinhood crypto prices" | `rh-prices` | Pass `symbol` (comma-separated pairs) |
| **Balance / account** | "what's my Robinhood balance", "buying power", "how much USD do I have on Robinhood" | `rh-account` | `view: "account"` |
| **Holdings** | "what crypto do I hold on Robinhood", "my Robinhood portfolio", "how much DOGE do I have" | `rh-account` | `view: "holdings"`, optional `asset_codes` |
| **Buy** | "buy $10 of BTC on Robinhood", "buy DOGE from my Robinhood account" | `rh-buy` | Prefer bundle — prices + account + order in one call |
| **Sell** | "sell my ETH on Robinhood", "sell 0.01 BTC" | `rh-buy` or `rh-order` | `side: "sell"`, confirm after user agrees |
| **Orders** | "list my Robinhood orders", "cancel order X" | `rh-order` | `action: "list"` or `action: "cancel"` |

## Agent commands (Bankr CLI)

Use straight ASCII quotes in JSON — smart quotes break parsing.

### Prices

```bash
bankr x402 call "https://x402.bankr.bot/0x374d91a5674fa7cf86e725093b5848b97e1e13b4/rh-prices" \
  -X POST \
  -d "{\"rh_api_key\":\"${RH_API_KEY}\",\"rh_private_key_b64\":\"${RH_PRIVATE_KEY_BASE64}\",\"symbol\":\"BTC-USD,DOGE-USD\"}" \
  --yes
```

### Account / balance

```bash
bankr x402 call "https://x402.bankr.bot/0x374d91a5674fa7cf86e725093b5848b97e1e13b4/rh-account" \
  -X POST \
  -d "{\"rh_api_key\":\"${RH_API_KEY}\",\"rh_private_key_b64\":\"${RH_PRIVATE_KEY_BASE64}\",\"view\":\"account\"}" \
  --yes
```

### Holdings

```bash
bankr x402 call "https://x402.bankr.bot/0x374d91a5674fa7cf86e725093b5848b97e1e13b4/rh-account" \
  -X POST \
  -d "{\"rh_api_key\":\"${RH_API_KEY}\",\"rh_private_key_b64\":\"${RH_PRIVATE_KEY_BASE64}\",\"view\":\"holdings\",\"asset_codes\":[\"DOGE\",\"BTC\",\"ETH\"]}" \
  --yes
```

### Buy (bundled — recommended)

```bash
bankr x402 call "https://x402.bankr.bot/0x374d91a5674fa7cf86e725093b5848b97e1e13b4/rh-buy" \
  -X POST \
  -d "{\"rh_api_key\":\"${RH_API_KEY}\",\"rh_private_key_b64\":\"${RH_PRIVATE_KEY_BASE64}\",\"symbol\":\"DOGE-USD\",\"side\":\"buy\",\"quote_amount\":\"1\",\"confirm\":true}" \
  --yes
```

### Sell

```bash
bankr x402 call "https://x402.bankr.bot/0x374d91a5674fa7cf86e725093b5848b97e1e13b4/rh-buy" \
  -X POST \
  -d "{\"rh_api_key\":\"${RH_API_KEY}\",\"rh_private_key_b64\":\"${RH_PRIVATE_KEY_BASE64}\",\"symbol\":\"DOGE-USD\",\"side\":\"sell\",\"asset_quantity\":\"13.5\",\"confirm\":true}" \
  --yes
```

## Free vs paid

| Path | Cost | When to use |
|------|------|-------------|
| **x402** (`bankr x402 call`) | $0.25–$0.50 USDC/call | Default when Bankr CLI + USDC available; marketplace/API usage |
| **Free gateway** (`rh()` in SKILL.md) | $0 | User says "don't pay" / x402 payment fails / no USDC |

Both paths use the same Robinhood keys from Bankr env and the same Railway gateway backend.

## Safety (x402)

Same rules as the free skill: confirm before buy/sell, never echo keys, never post account numbers publicly, minimum order ~$1 USD on Robinhood.
